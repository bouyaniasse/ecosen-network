Informations sur la plateforme

Présentation du projet

EcoSen Network est une plateforme de surveillance et de gestion des déchets développée pour la ville de Dakar, Sénégal.

Elle permet aux citoyens, opérateurs et administrateurs de signaler, suivre et traiter les incidents environnementaux en temps réel.

Développé dans le cadre du projet de fin d'études à l'Institut Africain de Management (IAM).

Stack technique

🟢 Backend : Node.js / Express
🗄️ Base de données : MySQL (Railway)
🔐 Auth : JWT maison
🌐 Frontend : HTML5 / CSS3 / JS vanilla
📊 Graphiques : Chart.js 4
🚀 Déploiement : Railway
📦 Version : 2.0.0
